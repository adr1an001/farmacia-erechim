# Landing Page - Farmácia

Projeto de uma landing page moderna desenvolvida para uma farmácia, com foco em apresentação de serviços, contato rápido e experiência do usuário.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 📌 Funcionalidades

- Layout responsivo para diferentes dispositivos
- Botão de contato via WhatsApp
- Seção de apresentação da empresa
- Cards de serviços/produtos
- Design moderno e organizado

## 🎯 Objetivo do projeto

Criar uma presença digital simples e profissional para uma farmácia que possui pouca presença online, demonstrando conhecimentos em desenvolvimento front-end.
